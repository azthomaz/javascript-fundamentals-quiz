# javascript-fundamentals-quiz
This is my "DOMination Assignment: Build a JavaScript Fundamentals Quiz". 

In this project you will find how I have combined my HTML, CSS and JavaScript abilities into one JavaScript Fundamentals interactive Quiz.

The browser will initially display an opening "Start" box. Once kicked-off, the quiz starts and you have 5 questions to answer.

At the end of the quiz a "Score" box will display your final score. You can restart from this point.

To run this project you will require an IDE where you can add the project's files and start the HTML file with a LiveServer like extension.
